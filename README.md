# EcoVeraZ Website (BUILD MODE)

## Setup
1) Install dependencies:
```bash
npm install
```

2) Run dev:
```bash
npm run dev
```

Open http://localhost:3000

## Notes
- Design tokens: `design-tokens/tokens.css`
- Tailwind is locked to tokens in `tailwind.config.ts`
- Global disclosures are enforced in `components/layout/Footer.tsx`
